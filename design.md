# 영감의 서랍 디자인 가이드

## 디자인 방향

- **전체 톤** : 미니멀, 여백 중심, 차분하고 감성적. 문장 자체가 주인공이 되도록 절제된 화면.
- **형태** : 각진 모서리를 지양하고, 전반적으로 **둥글게 라운딩된(rounded)** 형태를 사용한다. 카드, 버튼, 입력창, 모달 등 주요 요소 모두 모서리를 둥글게 처리해 부드럽고 따뜻한 인상을 준다.

## 컬러 팔레트

| 역할 | 컬러 이름 | HEX |
| --- | --- | --- |
| Primary | Soft Indigo Purple | `#6E6BA8` |
| Secondary | Lavender Mist | `#A7A3D8` |
| Accent | Periwinkle | `#7E8CE0` |
| Background | Warm Ivory | `#FAF8F5` |
| Text | Deep Navy | `#2C314A` |

### 컬러 사용 가이드

- **Primary (`#6E6BA8`)** : 주요 버튼, 핵심 인터랙션 요소, 강조가 필요한 영역
- **Secondary (`#A7A3D8`)** : 보조 버튼, 비활성/서브 요소, 부드러운 강조
- **Accent (`#7E8CE0`)** : 포인트 컬러. '소장하기' 버튼, 활성 상태, 링크 등 시선을 끌어야 하는 곳에 절제해서 사용
- **Background (`#FAF8F5`)** : 전체 배경. 따뜻한 아이보리 톤으로 눈의 피로를 줄이고 감성적인 분위기 형성
- **Text (`#2C314A`)** : 기본 텍스트 컬러. 순수 검정 대신 딥 네이비를 사용해 배경과 부드럽게 어우러지게 함

---

## 라운딩(모서리) 가이드

각진 느낌을 피하기 위해 요소별로 아래 기준을 적용한다. (Tailwind CSS 기준 예시)

- **카드** : 크게 둥글게 — `rounded-2xl` (약 16px) 이상
- **버튼 / 입력창** : `rounded-xl` (약 12px) 또는 `rounded-full`(알약형 버튼)
- **모달 / 시트 / 큰 컨테이너** : `rounded-2xl` ~ `rounded-3xl`
- **작은 태그 / 칩** : `rounded-full`

> 일관성을 위해 모든 인터랙션 요소는 둥근 모서리를 기본으로 하며, 직각 모서리(rounded-none)는 사용하지 않는다.

## 타이포그래피

- 본문 및 UI 텍스트는 미니멀한 톤에 맞는 깔끔한 한글 폰트 사용
- **카드 생성 시 사용자가 폰트를 선택**할 수 있으며, 선택지는 다음과 같다. (모두 Google Fonts)
  - 고딕 → Nanum Gothic
  - 명조 → Gowun Batang
  - 필기체 → Nanum Pen Script
  - 손글씨 → Hi Melody

---

## 전반 원칙

- 충분한 여백으로 답답하지 않게, 문장이 숨 쉴 공간을 확보한다.
- 컬러는 위 팔레트 내에서만 사용하고, 과한 그라데이션이나 강한 그림자는 지양한다.
- 그림자가 필요할 경우 옅고 부드럽게(soft shadow) 처리해 둥근 형태와 어울리게 한다.
