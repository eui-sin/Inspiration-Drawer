# 영감의 서랍 PRD

## 서비스 개요

- **서비스명** : 영감의 서랍
- **한 줄 설명** : 떠오르는 단상과 감명 깊은 문장을 나만의 카드로 소장하는 영감 아카이빙 플랫폼
- **사용 기술** : Next.js + Tailwind CSS
- **디자인 톤** : 미니멀, 여백 중심, 차분하고 감성적 (문장 자체가 주인공이 되도록 절제된 톤)

---

## 배경 / 문제 정의

떠오르는 단상이나 아이디어, 감명 깊은 문장을 만나는 순간은 많지만, 이를 **나만의 감성으로 아카이빙하고 가볍게 공유할 전용 공간이 없다.** 그 결과 기록이 메모 앱, 캡처, SNS 등으로 파편화되고 결국 휘발된다.

**원인**

- 기본 메모장은 감성적 몰입감이 떨어져 다시 열어보게 만드는 매력이 부족함
- 떠오르는 생각이나 문장을 개인 SNS에 공유하기에는 부담스러움
- 오로지 단상과 문장만을 다룰 수 있는 전용 공간이 부재함

---

## 프로젝트 목표

- 단상과 문장을 **'카드 이미지'** 형태로 소장하여 기록의 몰입도를 높인다.
- 메모장처럼 휘발되지 않고, 다시 열어보고 싶은 개인 영감 아카이브를 만든다.

---

## 대상 유저

떠오르는 생각이나 좋은 문장을 기록하고 싶지만, 메모장은 다시 열어보지 않아 기록이 휘발되는 게 아쉬운 20~30대.

---

## 핵심 기능 (MVP)

1. **카드 이미지화**
   문장을 입력하면 배경 컬러 선택 또는 이미지 등록을 통해 카드 형태로 시각화한다.
   - 카드의 폰트를 4가지 스타일 중 선택할 수 있으며, 각 스타일은 아래 웹폰트로 적용된다.
     - 고딕 → Nanum Gothic
     - 명조 → Gowun Batang
     - 필기체 → Nanum Pen Script
     - 손글씨 → Hi Melody

2. **갤러리형 아카이브**
   소장한 카드를 갤러리로 모아보고 다시 감상할 수 있는 개인 보관소를 제공한다.
   - 카드는 **생성 날짜별로 묶어서** 정리해 보여준다. (예: 날짜 헤더 아래에 해당 날짜에 만든 카드들이 모이는 방식)

3. **오늘의 문장 큐레이션**
   명언·영화 대사·책·시·칼럼 등 다양한 장르의 문장을 매일 한 편씩 제공한다.
   - 오늘의 문장 영역에는 **'소장하기'** 버튼을 두어, 누르면 해당 문장이 내 카드(아카이브)로 저장되도록 한다.

---

## MVP 범위 / 향후 확장

**MVP에 포함**

- 카드 생성(문장 입력 + 배경 컬러/이미지 선택)
- 갤러리형 아카이브 보기
- '오늘의 문장' 큐레이션 (사전 준비한 문장 데이터 기반, AI 불필요)
- 데이터 저장은 브라우저 로컬 저장(localStorage)으로 시작

**MVP 이후 (2차)**

- 공유 피드 + 공감 기능 등 소셜 인터랙션
- 회원 인증 및 데이터베이스 연동(Supabase 등)
- 로컬 저장 데이터를 DB로 이전

---

## 기술 참고 사항

- '오늘의 문장'은 미리 큐레이션한 문장 데이터(JSON 등)를 날짜별로 노출하는 방식으로 구현 → AI/외부 API 불필요
- MVP 단계에서는 별도 백엔드 없이 프론트엔드(Next.js) + 로컬 저장만으로 완결 가능
- 카드 폰트(Nanum Gothic / Gowun Batang / Hi Melody / Nanum Pen Script)는 모두 Google Fonts에서 제공되는 한글 웹폰트로, `next/font/google` 또는 link 태그로 불러와 적용
- 카드 저장 시 생성 날짜(타임스탬프)를 함께 기록하여, 갤러리에서 날짜별 그룹핑에 활용
